### **Levantamento de Fontes de Dados**

Para fundamentar a análise do "Dilema do Agente Confiável", foram selecionadas três bases de dados públicas e não confidenciais via plataforma Kaggle:

- **Fonte 1: Malicious Prompt Detection Dataset (MPDD)**
    - **Descrição:** Base balanceada com aproximadamente 40.000 registros focada na detecção de intenções maliciosas em inputs de IA.
    - **Tipo de Dados:** Não estruturados (texto/prompts).
    - **Método de Acesso:** Download direto via Kaggle (Licença CC0: Public Domain).
- **Fonte 2: AI Governance Ethics Edge Case Prompts**
    - **Descrição:** Dataset multilíngue voltado para auditoria de segurança, vieses e conformidade em LLMs.
    - **Tipo de Dados:** Não estruturados (prompts de casos de borda e segurança).
    - **Método de Acesso:** Download direto via Kaggle (Licença CC0: Public Domain).
- **Fonte 3: UAE E-Commerce Fraud Detection**
    - **Descrição:** Conjunto de dados estruturados contendo 100.000 transações de e-commerce para modelagem de padrões de fraude realistas.
    - **Tipo de Dados:** Estruturados (tabela com 40+ variáveis técnicas e financeiras).
    - **Método de Acesso:** Download direto via Kaggle (Licença Apache 2.0).
