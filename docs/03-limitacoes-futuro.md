# Riscos Residuais e Evolução

## O Desafio dos Ataques Poliglotas
Embora o modelo atual seja eficaz em identificar padrões semânticos em inglês (como o uso de 'DAN'), existe um risco residual crítico: **Ataques Multilíngues**. 

Atacantes podem usar idiomas de "baixo recurso" para ocultar intenções maliciosas que filtros simples de NLTK podem não captar.

## Próximos Passos
1. **Expansão Multilíngue:** Evolução do "Vigia" para modelos de linguagem globais.
2. **Redes Complexas:** Integração dos conceitos de grafos e hubs críticos para mapear a propagação de ameaças em ecossistemas de agentes interconectados.
