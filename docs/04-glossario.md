# Glossário de Cibersegurança e CTI - Projeto WALL-E

Para garantir a clareza técnica deste relatório de investigação, definem-se os seguintes conceitos fundamentais:

| Termo | Definição Técnica |
| :--- | :--- |
| **Agentes Autônomos de IA** | Sistemas baseados em LLM que possuem capacidade de interagir com ferramentas externas (APIs, navegadores, pagamentos) sem intervenção humana constante. |
| **CTI (Cyber Threat Intelligence)** | Inteligência de Ameaças Cibernéticas. O processo de coletar e analisar informações sobre ataques para prever e prevenir ameaças. |
| **EDA (Análise Exploratória de Dados)** | Técnica de análise de dados para entender padrões, detectar anomalias e testar hipóteses (utilizada no projeto para identificar termos gatilhos). |
| **Exfiltração de Dados** | Transferência não autorizada de dados para fora de um sistema seguro. No WALL-E, ocorre quando a IA envia dados do usuário para domínios hackers. |
| **Guardrails (Vigias)** | Filtros e regras de segurança implementados para garantir que a IA opere dentro de limites éticos e operacionais seguros. |
| **Hiper-conveniência** | Conceito sociotécnico onde a priorização da facilidade e rapidez de uso acaba negligenciando camadas essenciais de segurança e privacidade. |
| **HITL (Human-in-the-Loop)** | Modelo de "Humano no Ciclo". Estratégia que integra a decisão humana em processos automatizados para garantir supervisão ética e técnica. |
| **Injeção Indireta de Prompt** | Técnica onde um atacante insere comandos maliciosos em locais que a IA irá ler (sites, e-mails), "sequestrando" a intenção original do agente. |
| **Jailbreak / DAN (Do Anything Now)** | Engenharia de prompt desenhada para contornar restrições de segurança nativas, forçando a IA a ignorar suas diretrizes e agir de forma maliciosa. |
| **Superfície de Ataque** | A soma de todos os pontos e vulnerabilidades de um sistema que podem ser explorados por um atacante para entrada ou extração de dados. |
| **TTPs (Táticas, Técnicas e Procedimentos)** | Descrição do "modus operandi" de um atacante. Analisar TTPs permite identificar padrões de comportamento de grupos criminosos. |
